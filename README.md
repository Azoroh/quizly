# Quizly

Quizly is a modern quiz web application designed to make answering questions feel fast, focused, and engaging. It provides a clean interface, timed quizzes, real time progress tracking, and score feedback to create a simple but motivating learning experience.

The app is built with performance and usability in mind, allowing users to quickly start a quiz, move through questions smoothly, and challenge themselves to beat their highest score.

## 🚀 Live Demo

👉 https://jay-quizly.netlify.app/

## Features

- Timed quiz sessions
- Real time progress bar
- High score tracking
- Clean and minimal user interface
- Smooth micro interactions and transitions
- Fully responsive design

## Tech Stack

- HTML
- CSS
- JavaScript
- React (if applicable)

## How It Works

1. User lands on the start screen and can view their highest score.
2. Clicking the start button begins the quiz session.
3. Each question is displayed with multiple answer options.
4. A timer and progress bar guide the user through the quiz.
5. At the end, the final score is shown and the high score may be updated.
